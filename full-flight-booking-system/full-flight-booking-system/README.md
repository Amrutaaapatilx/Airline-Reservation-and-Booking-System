
# Full Flight Booking System

## Features
- User Signup
- Login
- Admin Dashboard
- Add Flights
- Flight Booking
- Passenger Management
- Seat Allocation
- Payment Calculation
- PostgreSQL Integration
- JSP Frontend
- Spring Boot Backend

## Default Admin
admin@gmail.com

## Run

1. Create PostgreSQL database:
CREATE DATABASE flightdb;

2. Update application.properties

3. Run:
mvn spring-boot:run

4. Open:
http://localhost:8080
