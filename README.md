# Airline-Reservation-and-Booking-System
A robust, Java-based Airline Reservation and Booking System . This application manages flight schedules, passenger bookings, and ticket reservations, leveraging an automated build pipeline for seamless development.
## Features
- User Signup
- Login
- Admin Dashboard
- Add Flights
- Flight Booking
- Passenger Management
- Seat Allocation
- Payment Calculation
- PostgreSQL Integration
- JSP Frontend
- Spring Boot Backend

## Run

1. Create PostgreSQL database:
CREATE DATABASE flightdb;

2. Update application.properties

3. Run:
mvn spring-boot:run

4. Open:
http://localhost:8080
